Japanese Object Detection & Translation

Overview

A computer vision web application that detects objects from images and translates its labels to Japanese with Furigana pronunciation guides.

Goal

To help language learners efficiently build better vocabulary while interacting with the real-world.

Screenshots

Features

  • Real-time object detection using DETR model
  • Automatic translation to Japanese
  • Furigana (hiragana) pronunciation guides
  • Mobile-friendly Progressive Web App
  • Camera integration for on-the-go learning

Future Improvements

  • Offline mode with cached translations
  • Multiple language support
  • Speech synthesis for pronunciation
